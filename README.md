# IBM-UI

## Requirements:

* nodejs: 16.7.0 (>15)

* data for backend:

Download and unzip <a href='https://drive.google.com/drive/folders/1MV5lcDlRgiRCtnRBm-KYT7xZV-WXloba?usp=sharing'> this </a> and move it into folder IBM-UI/backend

## Run;

### Frontend:

``` 
#from /IBM-UI
cd demo
npm install 
npm start
```

### Backend:

```
#from /IBM-UI
cd backend
```
